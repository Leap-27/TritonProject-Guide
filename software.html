Triton Project Software

Overview

The Triton will primarily be using the Leap Motion sensor for user input. Builders of the Triton will have the liberty to add whatever sensors they want to the headset but must be wary of the additonal weight that it can add.

Triton AR Launcher

One of the project goals of Triton was to stray away from using Unity. The Triton will be using an independent launcher built with Electron.js in order to launch AR content onto the headset. The Triton Project is primarily aiming to develop with the use of JavaScript, Three.js, and Leap.js.
